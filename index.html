    Projects

    3D Prosthetic Reconstruction

    Project Goals

    The project aimed to address prosthetic device accessibility in resource-constrained environments, leveraging computer vision and photogrammetry. The primary goal was to develop a cost-effective solution capable of accurately sizing prosthetic sockets from corresponding images, empowering communities with limited design equipment.

    Project Process

    My role involved developing a deep learning model for object detection and size estimation to tackle sizing challenges in the prosthetic socket. My work also involved generating DXF files with precise measurements all over the contour of interest, and can be leveraged in 3D CAD tools for further automation. Subsequently, I actively contributed to a collaborative effort with the Pittsburgh AiPEX lab, exploring 3D reconstruction algorithms and advancing an AI system for generating customized prosthetic designs, further enhancing accessibility.

    Arco: Sound Synthesis Server

    Project Goals

    In the Arco project, my objective was to showcase its capabilities by writing code to recreate demo music scores from the Resound repertoire. This task required proficiency in C++, Linux, Python, and Serpent (a python-like language developed by Professor Dannenberg for real-time systems), demonstrating Arco's prowess in a practical context.

    Project Process

    My role involved coding to illustrate Arco's functionalities and recreate music scores. This required a strong command of C++, Linux, Python, and Serpent. The collaborative effort contributed to showcasing Arco's capabilities in generating demo music scores aligned with the Resound repertoire.

    Livestock Biometrics

    Project Goals

    The goal was to enhance cattle identification and verification using deep learning, overcoming limitations of traditional methods and scalability challenges in classification neural networks.

    Project Process

    I led the design and training of a Siamese Network, addressing scalability issues with a contrastive loss function. Metrics evaluation included false acceptance rate, false rejection rate, and equal error rate. The successful implementation emphasized the need to optimize operational aspects for deploying deep learning solutions, fostering their commercialization and widespread adoption.

    Piano Transcription

    Project Goals

    The goal of this project was to develop a system to transcribe audio into music scores and MIDI files by re-implementing the work of Taegyun Kwon, Dasaem Jeong, and Juhan Nam.

    Project Process

    My role included setting up the cloud platform for training. This involved familiarity with Linux and Nvidia's CUDA toolkit for GPU acceleration. I was also involved in the training of the model and the development of a framework to evaluate the impact of label smoothing on the model's bias on different note states.
